# 股票趋势练习网站需求文档

## 📋 项目概述
**项目名称**：股票趋势练习网站 
**核心功能**：以特定日期为分界点，分别展示股票的历史走势和未来走势

## 🎯 核心功能需求

### 1. 股票选择与时间分界点设置
- **股票代码输入**：支持输入股票代码，提供自动补全功能
- **日期选择器**：精确到日的分界点设置,支持手动输入
- **数据验证**：验证股票代码有效性和日期合理性

### 2. 分界点前走势图表
- **K线图展示**：显示选定日期前的历史价格走势，支持MA、MACD等指标叠加
- **时间周期选择**：支持日线、周线、月线切换
- **成交量副图**：配合显示交易量变化，叠加MAVOL5、MAVOL10、MAVOL100移动成交量指标
- **KDJ副图**：配合显示KDJ变化，包含K线、D线、J线
- **时间轴缩放**：支持图表时间范围调整
- **交互功能**：鼠标悬停显示详细数据点

### 3. 分界点后走势图表
- **复用K线图功能**：复用分界点前走势图表的K线图展示功能
- **技术指标**：支持MA、MACD等指标叠加
- **预测区间**：显示可能的波动范围
- **关键标记**：标记重要价格水平

### 4. 对比分析视图
- **并排显示**：两个时间段走势的对比展示
- **差异分析**：直观显示两个时期的差异
- **统计对比**：关键指标的并行比较

### 5. 数据统计面板
- **关键指标**：涨跌幅、最高价、最低价、平均价
- **交易统计**：成交量、换手率等数据
- **时间段对比**：两个时间段的统计对比

## 🛠️ 技术方案

### 前端技术栈
- **框架**：React 18 + TypeScript
- **UI组件**：shadcn/ui（基于Tailwind CSS）
- **图表库**：Recharts（专业的图表可视化）
- **构建工具**：Vite

### 数据源
- **股票API**：使用AKShare开源金融数据接口库
- **数据获取**：Python后端服务通过AKShare获取A股实时和历史数据
- **数据格式**：JSON REST API
- **缓存机制**：Redis缓存减少数据请求，提升响应速度
- **数据特性**：支持A股、港股、美股、期货、基金等多种金融数据

## 🎨 设计风格
- **主题**：深色主题，科技金融风格
- **配色**：蓝色调科技感配色
- **布局**：左侧控制面板，右侧双图表展示区
- **交互**：玻璃态卡片设计，响应式布局

## 📊 开发计划（7个阶段）
1. 实现基础项目结构和主页面布局框架
2. 开发股票代码输入和日期选择器控制面板  
3. 集成股票API数据获取和解析功能
4. 实现分界点前历史走势K线图表组件
5. 开发分界点后未来走势趋势图表组件
6. 创建统计信息面板和对比分析功能
