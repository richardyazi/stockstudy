## Product Overview

更新股票K线图技术指标展示设计，参考同花顺等专业股票软件的界面布局，制定详细的K线图区域技术指标展示规范。

## Core Features

- K线主图区域设计（价格走势展示）
- 副图区域布局（技术指标展示）
- MACD指标展示方式
- KDJ指标展示方式  
- 成交量指标展示方式
- 技术指标切换控件
- 图表交互控件设计
- 响应式布局适配

## Tech Stack

- 前端框架：React + TypeScript
- 图表库：ECharts / TradingView Widget
- UI组件库：Ant Design / Material-UI
- 状态管理：Redux Toolkit / Zustand
- 样式方案：Tailwind CSS / Styled Components
- 图表数据处理：lodash / date-fns

## Design Style

采用专业的金融数据展示风格，参考同花顺、东方财富等专业股票软件的设计语言。整体界面采用深色主题，确保长时间使用的舒适性。图表区域采用模块化设计，主图区域突出K线展示，副图区域灵活展示各类技术指标。交互控件采用悬浮式设计，不遮挡图表内容。

## 界面布局设计

- **顶部工具栏**：时间周期切换、指标选择、绘图工具
- **主图区域**：K线图、均线系统、价格标签
- **副图区域**：成交量、MACD、KDJ等技术指标（可堆叠显示）
- **右侧信息面板**：实时价格、涨跌幅、成交量等关键数据
- **底部时间轴**：可拖拽缩放的时间选择器

## 技术指标展示规范

- **MACD指标**：DIFF线、DEA线、MACD柱状图，红绿配色区分涨跌
- **KDJ指标**：K线、D线、J线三条曲线，超买超卖区域标注
- **成交量指标**：红绿柱状图，与K线颜色同步，叠加均线
- **均线系统**：MA5、MA10、MA20、MA60，不同颜色区分
